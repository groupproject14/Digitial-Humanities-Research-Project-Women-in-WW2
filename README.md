# Digitial-Humanities-Research-Project-Women-in-WW2
Here is our website and spreadsheet investigating perspectives on women in world war 2 for the University of Southampton.
Our original data was collected from 150 images found on the imperial war museum website, here: https://www.iwm.org.uk/collections/search?query=&pageSize=30&media-records=records-with-media&filters%5BperiodString%5D%5BSecond%20World%20War%5D=on&filters%5BtermString%5D%5Bwomen%5D=on&style=image&page=8
they were collected and used under the Creative Commons Liscence attatched to this repository.
For contextual information and ethical guidelines on the data attached please read the website stored in this repository
